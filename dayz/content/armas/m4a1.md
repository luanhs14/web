# M4-A1

![M4-A1](https://via.placeholder.com/800x400?text=M4-A1)

## Descrição

A **M4-A1** é uma carabina modular de calibre 5.56x45mm NATO, amplamente reconhecida como uma das armas mais versáteis em DayZ. Sua plataforma altamente customizável permite adaptação para qualquer situação de combate.

## Características

| Especificação | Valor |
|--------------|-------|
| **Calibre** | 5.56x45mm |
| **Capacidade** | 30 cartuchos (padrão) |
| **Modo de Tiro** | Semi-automático / Automático |
| **Alcance Efetivo** | 400-500m |
| **Dano** | Médio-Alto |
| **Recuo** | Baixo |

## Munição Compatível

- 5.56x45mm NATO
- STANAG 30 rounds
- STANAG 60 rounds (Coupled)

## Sistema de Acessórios

A M4-A1 é conhecida por sua extrema modularidade:

### Trilhos (Rails)
- **RIS Handguard** - Adiciona 4 trilhos Picatinny
- **M4 Plastic Handguard** - Básico, sem trilhos

### Miras Ópticas
- **ACOG Optic** - Mira 4x ideal para média/longa distância
- **Red Dot Sight** - Combate rápido
- **Holographic Sight** - Versatilidade
- **ATOG Scope** - 6x magnificação

### Coronhas
- **M4 Collapsible Stock** - Ajustável
- **M4 Fixed Stock** - Fixa, maior estabilidade
- **M4 MP Stock** - Leve, mais mobilidade

### Acessórios de Cano
- **Suppressor 5.56** - Reduz ruído significativamente
- **Compensator** - Reduz recuo vertical
- **Muzzle Brake** - Reduz recuo horizontal

### Outros
- **Tactical Flashlight** - Iluminação
- **Laser Sight** - Melhora mira hip-fire
- **Vertical Grip** - Reduz recuo

## Vantagens

✅ Recuo extremamente controlável
✅ Alta precisão em todas as distâncias
✅ Máxima customização
✅ Munição comum (5.56mm)
✅ Excelente ergonomia
✅ Boa cadência de tiro

## Desvantagens

❌ Requer muitos acessórios para máximo potencial
❌ Dano ligeiramente menor que AK-101
❌ Sensível a sujeira (limpe regularmente)

## Dicas Profissionais

1. **Priorize a mira**: Uma boa mira óptica faz toda diferença
2. **Sempre use silenciador**: O ruído da M4 sem silenciador é muito alto
3. **Gerencie peso**: Cada acessório adiciona peso - escolha sabiamente
4. **Munição traçante**: Use para ajustar tiros de longa distância

## Comparações

### vs AK-101
- **M4**: Menos recuo, mais modular
- **AK-101**: Mais dano, mais resistente

### vs AUG
- **M4**: Mais acessórios disponíveis
- **AUG**: Mais compacta, built-in scope

## Manutenção

A M4-A1 requer limpeza regular:

- Use **Cleaning Kit** a cada 200-300 tiros
- Verifique o status da arma no inventário
- Arma suja = maior chance de travamento

## Status da Arma

| Condição | Descrição |
|----------|-----------|
| **Pristine** | Perfeitamente funcional |
| **Worn** | Funcionando bem, limpe em breve |
| **Damaged** | Começando a travar, limpe urgente |
| **Badly Damaged** | Trava frequentemente |
| **Ruined** | Inutilizável |

## PvP Tactics

### Engajamento 1v1
- Use semi-auto para conservar munição
- Mire para a cabeça
- Use cobertura entre os tiros

### Squad Play
- Coordene fogo de supressão
- Designe atiradores com ACOG para longas distâncias
- CQB com Red Dot para entradas em edifícios
